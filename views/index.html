Exercise Tracker

Create A New User 

POST   /api/exercise/new-user

Add Exercise 

POST   /api/exercise/add
Get user's exercise Log : 
GET api/exercise/log/{userId}?[from]&[to]&[limit]

{ } is required , [ ] is optional ,
from , to = date(yyyy-mm-dd) , limit = number
Example : https://example.glitch.me/log/56j34535jjhjnh4534?from=2018-10-10&to=2019-1-1&limit=3
